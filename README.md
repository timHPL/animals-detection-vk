# Детекция животных на фотографиях профиля
На фотографиях из выбранного альбома выделяются животные
## Набор данных
https://www.kaggle.com/datasets/antoreepjana/animals-detection-images-dataset
## Модель детектора
Cascade R-CNN (Resnet50)
## Технологии
* Python
* MMDetection
* OpenCV
* VK api
* Pytorch
## Результаты
mAP = 0.5795
